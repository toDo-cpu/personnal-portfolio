Abrorescence:

    assets: 
    Contient tous les fichiers hormis les htmls qui sont à la racines du site

    assets/css: 
    Contient tous les fichiers css, ils ont le même nom que la page html avec qui ils sont associé
    N.B: global.css est inclu partout, il contient le css global du site

    assets/js:
    contient l'unique fichier js du projet permmettant le changement du thème de couleur

    assets/font:
    contient les polices du site

    assets/img:
    contient toutes les images
